# JavaCollections
```
Данное приложение является лабораторной работой для курса разработки приложений на Java (тема - работа с коллекциями).
Программа считывает данные из файла data_diamond.txt, формируя при чтении объекты класса Diamond и помещая их в ArrayList<Diamond>. 
Потом данные фильтруются в соответствии с заданием и выводятся в файл и консоль.
```
